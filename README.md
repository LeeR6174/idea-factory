# 💤 SiestaHub | 昼寝記録・タイマー管理アプリ

昼寝専用のタイマーをTOPページに配し、アラーム完了時にバイブレーション・電子音でお知らせする、モダンで高機能な昼寝ログ記録アプリです。
Flask + SQLiteでのしっかり保存に加え、サーバーを立ち上げずにHTMLファイルをそのままブラウザで開いても自動的にLocalStorageへフォールバックして動く「ハイブリッド・アーキテクチャ」を採用しています。

---

## 🌟 主な機能

### 1. 昼寝専用タイマー（TOP配置）
*   **プリセット**: パワーナップ(10分)、リフレッシュ(15分)、シエスタ(20分)、ディープ(30分)を1タップで切替。
*   **カスタムタイマー**: 1分〜90分のスライサー付き。
*   **SVG円形プログレスリング**: 直感的でなめらかに減少する円形のタイマーカウント。
*   **バックグラウンド対応（ドキュメントタイトル連動）**: タブを閉じずに他のアプリをしていても、ブラウザのタイトル（タブ名）に残り時間が表示されます。

### 2. バイブレーション & アラーム音お知らせ
*   **📳 バイブ通知**: `navigator.vibrate` (Web Vibration API) に対応。アラームの終了を心地よい振動のパターンでお知らせ。
*   **🔊 合成アラーム音**: `Web Audio API`を使用し、ブラウザ上でリアルタイムに和音(ミ・ラ)を合成して鳴らします（外部のMP3や音声アセットに依存しないため、ネットオフラインでも確実にアラーム音が鳴ります）。
*   **テストボタン**: 実機でバイブレーションが正しく機能するか瞬時に確認できます。

### 3. スッキリ度の評価 & 記録
*   **詳細記録**: 終了時に「場所（ベッド、ソファ、デスク、他）」、「スッキリ度 ⭐1〜5段階」、「メモ（夢の有無、すっきり度合い）」を入力・追加。
*   **手動入力**: タイマーを使わずに昼寝を済ませてしまった場合も、過去の昼寝を後付けで追加記録できます。

### 4. グラフ & 統計情報の可視化 (Chart.js)
*   **スタッツパネル**: 総昼寝回数、平均スッキリ度、平均昼寝時間を上部に瞬時に集計。
*   **直近の昼寝時間推移**: 過去7回分の昼寝時間をスタイリッシュな棒グラフで表示。
*   **スッキリ度割合（ドーナツチャート）**: 寝起きのコンディション比率を分析表示。

### 5. データポータビリティ & デモ
*   **📤 エクスポート**: 全ログを `JSON` ファイル形式でワンクリックダウンロード可能。
*   **📥 インポート**: バックアップしたJSONファイルを読み込み、他のブラウザやデバイスから復元可能。
*   **✨ デモ生成**: お試し用のダミーデータをボタン一つで8件自動挿入。一瞬で美しいグラフや統計を確認できます。

### 6. ダークモード完全対応
*   寝不足のデリケートな目にも優しい高コントラストの「ダークモード（デフォルト）」と心地いい「ライトモード」を目玉のトグルボタンで瞬時に切り替え可能。

## 🛠️ 技術スタック
*   **Backend**: Flask (Python 3) + SQLite3 (軽量で構築不要)
*   **Frontend**: HTML5, Vanilla JS (ES6), Tailwind CSS (スタイリング), Chart.js (グラフ)
*   **API / Features**: HTML5 Vibration API, Web Audio API

## 🚀 起動方法

### 必要なライブラリのインストール
リポジトリ直下にて以下のコマンドで依存関係(Flask)を追加・インストールします。
```bash
pip install -r requirements.txt
```

### サーバーの起動
```bash
python app.py
```
起動完了後、ブラウザで [http://127.0.0.1:5000](http://127.0.0.1:5000) にアクセスします。
データベース (`naps.db`) が自動的にローカルに生成され、データの保存が開始されます。

#### 💡 スタンドアロン動作（サーバーを起動したくない場合）
`templates/index.html` ファイルをブラウザでそのままドラッグ＆ドロップして開くだけでも動作します。データベースの接続が切れていることを検知すると、ヘッダーに Indigo 色の「**LocalStorage 保存中**」バッジが自律点灯し、すべてのデータがブラウザの LocalStorage に自動保存・読み込みされます。
これにより、サーバー設置なしの静的サイトや、完全なオフラインポータブルツールとしても、1ファイルを共有するだけでどこでも使えるよう配慮されています。
